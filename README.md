## python_pi

```
git clone https://github.com/b-kaxa/python_pi.git
cd python_pi
bower install
python pi_search.py
```

* pythonにおける円周率10000桁から任意の数字を探す誰得プログラムです

* 大学の授業の課題で作りました

## 以下最終課題提出用

* 任意の数字の最終位がスコアになり、計三回数字を入力してスコアを競い合う簡単なゲームに改造しました
